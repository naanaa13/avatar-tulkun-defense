## Deskripsi
Game 2D defense shooter menggunakan Python + Pygame.  
Pemain mengendalikan Tulkun untuk bertahan dari serangan kapal RDA, terinspirasi film Avatar: The Way of Water.

## Cara Menjalankan
1. Install Pygame: `pip install pygame`
2. Jalankan file utama: `python ocean_tulkun_game.py`

## Fitur Utama
- Kontrol pemain (WASD/Arrow + Space)
- Sonic Wave vs Harpoon
- Koleksi Eywa Seed
- Efek visual glow & particle explosion
- Sound effects + pergantian musik dramatis saat game over ("The Songcord")
- High score tersimpan

## Struktur Folder
- `ocean_tulkun_game.py` → kode utama
- `assets/` → gambar dan file audio
